## 特性

1. 启用后可以为 ChatLuna 提供文件中转服务，让模型可以访问到文件地址

## 部署

在 Koishi 插件市场搜索 `chatluna-storage-service`，安装后启用即可。

**插件依赖 ChatLuna，请确保在 ChatLuna 启动后再启用本插件。**
